# Kuran Kaşifi ✦

Kur'an-ı Kerim'in tüm **6.236 ayetini** Arapça metin, Türkçe meal ve kelime kelime üzerine gelince çeviri özelliğiyle keşfeden interaktif bir uygulama.

🌐 **[Canlı Demo](https://sedatkacar56.github.io/kuran-kasifi)**

---

## ✨ Özellikler

- **Rastgele Ayet** — Tek tıklamayla Kur'an'ın herhangi bir ayetini keşfet
- **Arapça Metin** — Amiri yazı tipiyle güzelce işlenmiş Osmanlı hattı Arapça
- **Kelime Üzerine Gelince İpucu** — Herhangi bir Arapça kelimenin üzerine gelerek okunuşunu ve Türkçe anlamını gör
- **Türkçe Meal** — Diyanet İşleri meali her ayetle birlikte gösterilir
- **Sure Bilgisi** — Sure adı, numarası, ayet referansı ve 114 surenin tamamı için el yazısıyla hazırlanmış konu özetleri
- **114 Surenin Tamamı** — Fâtiha'dan Nâs'a, her surenin tematik özeti mevcuttur

## 🛠️ Teknoloji

| Katman | Teknoloji |
|---|---|
| Arayüz | Saf HTML, CSS, JavaScript |

| Kur'an Metin API'si | [alquran.cloud](https://alquran.cloud/api) |

## 📡 Kullanılan API'ler

| API | Amaç |
|---|---|
| `api.alquran.cloud/v1/quran/quran-uthmani` | Tam Arapça Kur'an (Osmanlı hattı) |
| `api.alquran.cloud/v1/quran/tr.diyanet` | Diyanet İşleri Türkçe Meali |
| `api.qurancdn.com/api/qdc/verses/by_key/` | Kelime düzeyinde okunuş ve çeviri |

Tüm API'ler ücretsiz ve herkese açıktır. API anahtarı gerekmez.

## 🙏 Teşekkürler
